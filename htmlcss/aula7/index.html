Redimensione a tela para ver as variáveis **font-size** e **spacing** mudarem.

**Tamanho Atual do H1:** var(--font-size-h1)

Título Principal (H1)

Este texto base usa a variável **--font-size-p**. A margem inferior deste parágrafo e o padding desta seção são controlados pela variável **--spacing-element** e **--spacing-section**.

Em telas pequenas (padrão), o H1 é menor e o espaçamento é reduzido. Conforme a tela cresce, o Media Query sobrescreve as variáveis, e o layout se adapta.

Subtítulo de Exemplo

Note como a área de padding desta seção muda drasticamente quando você transiciona entre os diferentes tamanhos de tela definidos nos Media Queries (600px e 992px).
